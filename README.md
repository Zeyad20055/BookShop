# 📚 BookShop - Full Stack E-Commerce Platform

## 📌 Project Overview

BookShop is a complete Full Stack web application for managing and selling books online.

The project contains:
- Frontend built with React.js
- Backend built with Node.js & Express.js
- Database using MongoDB
- Authentication and authorization system
- Admin dashboard for managing books and users

The goal of this project is to provide a modern online bookstore experience with a complete backend API and responsive frontend interface.

---

# 🚀 Features

## 👤 User Features

- User Registration & Login
- Authentication using JWT
- Secure cookies handling
- Browse available books
- View book details
- Search books
- Add books to favorites/cart
- Responsive design for all devices


## 🔐 Admin Features

- Admin authentication
- Add new books
- Update book information
- Delete books
- Manage products
- Upload book images
- Control featured books
